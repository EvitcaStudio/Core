    Chronos

    The Chronos class
    A fully flexible interval/timer with chainable API

    Example:

    Example usage of this class
    // Create a timer and start it
const timer = new Chronos();
timer.start();
// Or
const timer = new Chronos().start();

// Call a function every n ms.
timer.every(5000, () => { console.log('call me every 5000ms'); }).start();

// Call a function at a specific time in this timer's duration
timer.at(3000, () => { console.log('Called at 3000ms'); });

// Create a timer with custom settings
const timer = new Chronos({
 duration: 5000,
 startAt: 1000,
 flow: 1,
 tickRate: 10,
 completionCount: 50,
 timeScale: 2,
 loop: true
});

    public off(pEventName: string, pCallback: function): this source

    Removes a callback from the event name specified. If no callback is passed then all existing callbacks are removed from the event

    Params:

    NameTypeAttributeDescription
    pEventName string

    The event name you want to remove the callback from
    pCallback function

    The callback to remove from this event

    Return:

    this

    The chronos instance

    Example:

    // Create the timer
const timer = new Chronos();

// Store a reference to the callbacks since we will need to pass them back to remove them
const startEvent = () => { 
 console.log('start'); 
};

const pauseEvent = () => { 
 console.log('paused'); 
};

// Attach events to the timer
timer.on('start', startEvent).on('pause', pauseEvent);

// Start the timer so the event `start` can call your attached callback.
timer.start();
// Pause the timer so the event `pause` can call your attached callback.
timer.pause();

// Remove the events
timer.off('start', startEvent);
timer.off('pause', pauseEvent);

// Call the events again to see that the events are no longer being listened to
timer.start();
timer.pause();
